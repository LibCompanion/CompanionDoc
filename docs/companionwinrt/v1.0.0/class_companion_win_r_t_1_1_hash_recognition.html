Detailed Description

This class provides a WinRT wrapper for the 'HashRecognition' functionality of the Companion framework.

Note 1: This is a minimum construction – it has to be extended to provide the same flexability as the Companion native code.

Note 2: Native code in interfaces and public inheritance are not possible in a Windows Runtime context (with very few exceptions). We can not mirror the plausible interface / abstract class 'ImageProcessing' for this wrapper.
