Object Detection

Feature Matching

This compontent is a interface for all object detection implementations for example Feature Matching or Template Matching based on OpenCV 3. This interface requires a Companion::Algorithm::Recognition::Matching implentation which is a image processing algorithm. To improve image processing results images can be scaled to different dimensions. Optional a shape detection implementation can be attached to get only Region of Interests (ROI).

/**
 * Constructor to create an matching algorithm implementation.
 * @param matchingAlgo Matching algorithm to use, for example feature matching.
 * @param scaling Scaling to resize an image. Default is full hd.
 * @param shapeDetection Shape detection algorithm to detect roi's in images (if not set the whole image will be searched).
 */
MatchRecognition(Companion::Algorithm::Recognition::Matching::Matching *matchingAlgo,
    Companion::SCALING scaling = Companion::SCALING::SCALE_1920x1080,
    Companion::Algorithm::Detection::ShapeDetection *shapeDetection = nullptr);
Supported Matching Algorithms
Feature Matching Companion::Algorithm::Recognition::Matching::FeatureMatching


Supported Shape Detection Algorithms
Rectangle Detection Companion::Algorithm::Detection::ShapeDetection

Image Hashing

For image Hash processing use the HashRecognition interface. This interface needs a fixed model size in px for searched models. To detect region of interests a shape detection must be set and at least the supported hashing processing for example LSH.

/**
 * Hash recognition construtor.
 * @param modelSize Model size in pixel.
 * @param shapeDetection Shape detection algorithm to detect ROI's.
 * @param hashing Hashing algorithm implementation, for example LSH.
 */
HashRecognition(cv::Size modelSize,
    Companion::Algorithm::Detection::ShapeDetection *shapeDetection,
    Companion::Algorithm::Recognition::Hashing::Hashing *hashing);
Supported Hashing Algorithms
Local sensitive hashing (LSH) Companion::Algorithm::Recognition::Hashing::LSH

Hybrids

A hybrid approach is implemented to search for image models (ROIs) and to verify roi with a feature matcher.

/**
 * Hybrid recognition constructor.
 * @param hashRecognition Hash recognition to use.
 * @param featureMatching Feature matching to verify recognized hashes.
 * @param resize Resize image factor from 1 to 100. 100 original scale 1 is 99 percentage smaller image.
 */
HybridRecognition(Companion::Processing::Recognition::HashRecognition *hashRecognition,
    Companion::Algorithm::Recognition::Matching::Matching *featureMatching,
    int resize = 100);
Supported Matching Algorithms
Feature Matching Companion::Algorithm::Recognition::Matching::FeatureMatching


Supported Hashing Algorithms
Local sensitive hashing (LSH) Companion::Algorithm::Recognition::Hashing::LSH
